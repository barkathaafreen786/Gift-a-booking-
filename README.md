# Gift A Scene - Booking Experience

A specialized web application for browsing and gifting unique entertainment experiences. Built with a focus on vibrant aesthetics and smooth user interactions.

## üåü Overview

This project is a booking platform designed for "Gift A Scene," allowing users to:
- Browse curated experiences (Pottery, Lazer Tag, Water Sports, etc.).
- Visualize what the experience looks like through high-quality imagery.
- Book an experience as a gift for someone else.
- Send a personalized message along with the booking.

The design emphasizes a premium, modern feel using a "Cool to Warm" color spectrum and responsive layouts.

## üöÄ Features

- **Dynamic Experience Listing**: Experiences are fetched from a backend API and rendered dynamically on the frontend.
- **Responsive Design**: Fully responsive layout that adapts to mobile, tablet, and desktop screens.
- **Interactive Global Styles**: deeply customized CSS variables for theming, including a signature "Cool to Warm" gradient.
- **Booking System**: A functional form to capture sender/recipient details and preferred dates.
- **Backend API**: A Node.js/Express server that manages experience data and handles booking requests.

## üì¶ How to Run

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/barkathaafreen786/Gift-a-booking-.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd Gift-a-booking-
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Start the server**:
    ```bash
    node server.js
    ```
5.  **View the application**:
    Open your browser and navigate to `http://localhost:3000`.
